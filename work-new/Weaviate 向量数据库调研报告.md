# Weaviate 向量数据库调研报告

## Weaviate 功能介绍

Weaviate 是一个开源的**向量搜索引擎**，结合了**向量数据库**和**机器学习模型**的能力，主要特点包括：

1. **语义搜索能力**
   - 支持自然语言查询，理解语义而非关键词匹配
   - 基于向量相似度进行内容检索
2. **多模态支持**
   - 支持文本、图像、音频等多种数据类型
   - 内置多种向量化模块（OpenAI、Hugging Face 等）
3. **AI原生架构**
   - 集成生成式AI（如GPT模型）
   - 支持问答生成、摘要提取等高级功能
4. **云原生设计**
   - 支持分布式部署和水平扩展
   - 提供托管云服务（Weaviate Cloud Service）
5. **混合搜索**
   - 结合向量搜索和传统关键词搜索
   - 支持元数据过滤和混合检索
